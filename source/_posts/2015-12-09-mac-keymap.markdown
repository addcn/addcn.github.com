# mac快捷键

| `快捷键` | 功能   | 
| ------------- | ------- | 
| `cmd + space`    | 输入法切换    | 
| `cmd + f3`    | 显示桌面    | 
| `cmd + 上下箭头`  | 上级／下级目录 | 
| `enter`       | 重命名文件   | 
| `cmd + c`     | 复制文件    | 
| `cmd +s`      | 保存      |
